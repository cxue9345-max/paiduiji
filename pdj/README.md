# 插件目录

## 排队插件

> 通过获取直播间弹幕信息，进行业务处理

### 功能

> 检测 **排队** 弹幕 把发送弹幕的人员加入排队列表
>
> 检测 **超级排 内容** 弹幕 把发送弹幕的人员加入排队列表，并且更加酷炫。

> 检测 **排队加空格** 弹幕 把发送弹幕的人员和空格后的内容加入排队列表

> 检测 **取消排队** 弹幕 把发送弹幕的人员删除排队列表
>
> 检测 **排队取消** 弹幕 把发送弹幕的人员删除排队列表

> 检测 **插队 1 456** 弹幕 把发送弹幕的管理员所说的文本 456，插入到1的位置。


> 自定义删除权限设置 根据坐标删除排队人员需要指定用户权限
>
> 发送 del1或者1del 可以删除排队列表中的第一位  数字是多少就删除多少位 

### 插件目录 

`./dograin`

> 核心实现使用开源项目   https://gitee.com/lazycat-xiao/dograin.git 获取弹幕信息

> 使用方式  window系统 双击nginx.exe  可自行设置开机启动 主要解压文件夹不要中文、会存在编码问题
>
> 测试 在游览器访问 http://localhost/dograin/

### **注意事项**

> **本插件为页面缓存，刷新游览器控件或重启obs 排队数据会丢失**
